
# Crownland Camping Best Spots

Static mobile‑first map showing crownland camping GPS spots.

## Features
- Interactive map with 185 camping pins
- Tap a pin → opens Google Maps navigation
- Works on mobile
- No API keys required

## Deploy to GitHub Pages

1. Create a new repository
2. Upload these files
3. Go to **Settings → Pages**
4. Select **Deploy from branch**
5. Choose **main / root**

Your website will be live at:

https://YOURUSERNAME.github.io/REPO/
